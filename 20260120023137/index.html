虎鲸笔记插件：恐龙工具箱

这是一个为 Orca Note (虎鲸笔记) 开发的增强插件集合，提供了一系列实用的增强工具，帮助你更高效地管理和组织笔记。

这是一只恐龙，路过的人必须夸它可爱。

安装与使用

  1. 下载 Release 中的 package.zip 文件。
  2. 解压后，将整个文件夹复制到 Orca Note 的插件目录。
  3. 在软件设置中刷新并启用插件。插件集内包含的所有子功能均可在“插件设置”中单独开启或关闭。

功能列表

1. 📝 一键格式化 (Format Block)

标准化当前块及子块的文本格式。自动修正中英文标点符号，优化中英文混排空格，规范化全角/半角字符。

2. 🎙️ VoiceNotes 同步

将 voicenotes.com 的录音笔记一键同步到虎鲸笔记中，自动归档到专属收件箱，支持增量与全量同步。

3. 📂 导入工具 (Import)

支持批量导入本地 Markdown 文件夹；支持将 CSV 表格数据智能导入并解析为笔记节点，可自动应用多模板标签。

4. 🧹 样式清除 (Remove Style)

快速清理富文本格式。支持一键移除行内样式（加粗、颜色）、将链接转为纯文本、删除空行，或将各种标题统一转为自动层级标题。

5. 🔃 块排序 (Sort Blocks)

智能对选中的多个同级块进行排序。

  • 多预设配置：支持在设置中创建无限个自定义排序预设（Profiles）。
  • 原生规则调序：支持通过上下箭头对不同类型块（任务、标题、媒体、空白块等数十种类型）的排序优先级进行灵活组合。
  • 智能菜单：如果只保留一个预设，右键一键即可秒速排序；如果有多个预设，右键菜单会自动变为子菜单供你选择具体的排序规则。

6. 📤 发布到 GitHub (Publish)

将笔记一键发布为 Markdown 博客文章。自动提取并上传文中图片到图床，自动生成 Frontmatter 元数据。

7. 🏪 插件市集 (Bazaar)

Orca 社区插件市场。在应用内即可浏览、安装、更新和卸载社区插件，支持自定义第三方插件源。

8. 🏷️ 快捷标签 (Quick Tag)

使用快捷键一键在光标处插入常用标签，并可自动附加指定的属性（如状态、优先级等）。

9. 🌳 标题层级整理 (Heading Tree) 🥰 SaXz2

智能识别现有笔记的 H1-H4 标题层级，重组文档树结构，消除错误的阶梯式缩进，并将普通文本自动归属到最近的标题下。

10. 🌐 网页助手 (Web Assistant)

提供内置沉浸式浏览器。支持智能提取网页元数据（如豆瓣图书、电影），右键一键将选中的网页内容转换为 Markdown 剪藏到笔记中。

11. 🧠 记忆复习 (SRS)

集成 FSRS 算法的高效记忆工具。提供问答和沉浸式阅读两种复习模式，支持为任意标签或查询结果开启复习漫游。

12. 🗺️ 脑图视图 (Mind Map)

一键将当前大纲笔记转化为直观的思维导图，支持自由拖拽、缩放与节点折叠，无缝切回文本视图。

13. 🪞 块嵌入子项增强 (Embed Children)

优化镜像块的展示体验。允许带有特定标签的镜像引用块，自动向下穿透并展示其包含的子块内容。

14. 🎲 随机漫步 (Random Walk)

知识库漫游工具。自动抓取特定标签下的内容进行无重复发牌，帮助你随机探索和温习过去的知识片段。

15. 📋 智能剪贴板注入 (Clipboard Injection)

支持将外部复制的专属 JSON 数据直接解析为原生的虎鲸笔记节点。支持携带标签、属性，并可自动转存远程图片到本地。

16. ⌨️ 编辑器扩展命令 (Editor Commands)

选中任意文本即可使用命令将其“复制为带有块引用的 JSON”，在其他页面粘贴即可快速建立双向链接。

17. 🛠️ 块工具箱 (Block Tools)

提供丰富的块操作快捷菜单。支持一键将子块归集到原引用源、在父节点下置顶/置底，以及在链接和引用格式间快速转换。

18. 🔒 隐私块 (Privacy Block)

一键将块设为隐私状态。不仅在视觉上有锁形提示，底层也会向 AI 隐藏敏感内容，有效保护日记隐私，且不影响正常的编辑书写体验。

19. 🌊 块流转 (Block Flow)

右键快速将选中的块发送、移动或引用至「今日日志」、「明日日志」或你指定的「全局收件箱」。

20. 📌 置顶块面板 (Pinned Blocks)

为高频笔记打上特定标签，即可在侧边栏面板中将其固定，一键点击即可随时唤回核心工作区。

21. 🪐 Arc 侧边栏 (Arc Tabs)

媲美 Arc 浏览器的现代化侧边栏。支持多空间隔离、固定标签页自由拖拽排序，以及自动记录去重今天的历史浏览，告别多页面混乱。

22. 📥 全局收集箱 (Roam Sidebar)

复刻 Roam Research 的右侧边栏全局收集交互。底座为完全持久化的原生大纲块，支持跨页面随意拖拽收集、大纲折叠管理，状态永不丢失。

23. 🔗 链接工具 (Link Tools)

为原生链接提供快捷转换选项。右键超链接即可一键转为原生块引用，或者在图钉（📌）引用格式间灵活切换。

24. 🌳 大纲导航树 (Block Nav)

全局侧边栏大纲导航。支持点击平滑跳转、快捷下钻阅读；提供实时本地检索框，搜索结果瞬间过滤。底部提供层级展开按钮：左键展开侧边栏大纲，右键无缝联动主编辑器执行对应层级的精准折叠展开，按住 Alt / Shift + 左键则同时展开大纲与主编辑器。

25. 🔗 收集引用 (Gather Refs)

聚合当前文档的所有反向引用。支持通过命令面板 (/gather) 或右键菜单，自动提取所有引用了当前文档的顶级节点（如页面或卡片），并利用 Markdown 语法批量去重插入到光标位置，生成整洁美观的 MOC (Map of Content) 目录树，同时避免反向链接面板污染。

26. 🧊 编辑器层级折叠 (Editor Fold)

精准控制主编辑器的视觉层级。提供“展开至此同级”右键菜单以及支持快捷键绑定的全局命令，自动截断深层结构的展示，保持工作区绝对整洁。

27. 🕸️ 局部关系图谱 (Local Graph)

全局侧边栏局部网络关系图谱。实时跟踪编辑器导航轨迹，动态渲染当前活动块与足迹网络的邻居网络。提供对标签节点、大纲树父子层级关系、常规引用的快捷过滤选项；支持右键点击边缘节点进行定向本地拓展或折叠收起；内置防剧烈晃动的物理阻尼控制，让知识漫游更加丝滑稳定。新增回放探索模式，点击幻灯片图标进入回放，足迹节点按时序逐步呈现，相机自动跟随；用户拖拽后自动暂停跟随，退出时自动恢复录制状态。

开发说明

本项目采用模块化架构：

  • src/lets-*：每个目录对应一个子插件，功能相互独立。
  • main.tsx：负责管理与动态加载所有启用的子插件。
