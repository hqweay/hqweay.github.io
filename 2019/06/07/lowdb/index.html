谈谈 lowdb 的使用

发表于 2019-06-07 更新于 2023-07-01 分类于 学习折腾 ， 配置与操作

LT;DR;

博主在使用 lowdb 时遇到了几个需求光靠 lowdb 提供的 API 无法实现。其实 lowdb 是基于 Lodash 的，可以使用 Lodash 提供的 API 实现自己的一些需求。比如下面提到的「数组删除元素」与「深查询」。

基本上查看下面的文档就行了：

  • lowdb 的 Github 仓库
  • Lodash 官方文档

PS：至 2019-06-07 日，配合 TypeScript 使用 lowdb 时，如果要使用 Lodash 的 API（当然，直接使用 Lodash 也同理。），即便你安装了 @types/lodash ，仍然有许多方法会报编译错误。这是因为声明还没覆盖完 lodash 的方法。使用时可以忽略这些错误或者设置 //@ ts-ignore 忽略编译报错。
