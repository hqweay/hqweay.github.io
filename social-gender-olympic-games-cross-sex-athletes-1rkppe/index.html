社会性别、奥运会、跨性别运动员

发表于 2024-08-04

问题：

能讲讲你怎么看待社会性别与跨性别运动员嘛？

一些社会性别理论的论证基于否定生物决定论，把生理性别和社会性别区分开来。

在这个思路下，似乎变性手术便不是一件值得鼓励的事。人就本 没有必要 通过物理的调整 让自己的身体与社会认知一致。尊重、感谢我们的身体。身体并没有错。

这么来看跨性别运动员的存在本身也是一种（适应制度的）扭曲。它将不同的人，以各种方式纳入到两种性别里来做一些竞技。

或许就干脆地立好规则，以物理意义上的性别作为分类…

可「物理意义上的性别」也没那么简单。怎么区分？染色体？睾酮素？

拥有 XXX 染色体的人表现为女性性状，也能正常生活…就把他们当作「非正常人」不做考虑嘛？

你想到什么？能讲讲吗？

讨论一：

让我们从数据管理的角度来看看「社会性别」。比如有一张表，其有 100 列，即 100 个属性，每一行代表一个人的基本特征，包括但不限于各项生理指标等。

其中 sex（生理性别）列的属性值在设计上只应该有 2 个——男、女。sex 列根据算法由其他几列计算而来。可是在第一版算法中，计算依赖的某些列的值缺失或异常，导致计算结果为 null，然而这一值在现有规则下是不允许为 null 的，（类比出生登记的性别、身份证上的性别）。于是第二版算法在第一版的基础上根据某些列的值做了异常处理（如根据「性染色体」、「性器官」等）。

所有的人参与社会生活到一定程度后，因各种原因（如 ① 发现自己的综合表现更符合另一个 sex 值的特征；② 认为另一个 sex 值更能在社会中占优势；③ 厌恶自己的当前身体，想要变得不同……）一部分人想要保留 sex 列的值，但是新增一列 gender（社会性别）；一部分人会想要修改 sex 列的属性，这样 sex 与 gender 就是一样的了，即使这些人的表现不符合该 sex 的通常表现……

接下来谈谈「跨性别运动员」，这些人被允许参与比赛，那大概率是满足了比赛参与规则，规则是人制定的。但是「跨性别运动员」是如何产生的呢？这就有点像社会性别的产生，类比即可。

任何运作良好的社会都是倾向于规则、约束、秩序，讨厌 null 值的，这个 null 的意思并不是「没有价值」、「无用的」，而是「未知」。
一部分 sex 有 null 值的人在积极地配合社会规则改造自己，一部分人想要改变自己的 sex 值，而另一部分人认为自己不应该被定义，宣称自己的 sex 值应该是 null。然而无论多少种情况，怎样设计改变，都不能让组成社会的所有个体满意。

「社会很单纯，复杂的是人」。这是现状，也将永远如此。

讨论二：

假定随着社会的发展会（应该）越来越包容，让更多不同的人纳入进「正常人」的范畴。（如性别的枚举值扩展）

运动会就是一个很好的实践场所。它的规则是设计出来的（社会的规则也是，但大概没那么明显）。运动会的规则可以改变（不像社会习俗那么难改变）。

那运动会（比如奥运会）作为「新老观念」冲突之处，如果能让跨性别运动员这回事得到很好的解释，社会观念也会得到很好的…发展。

现在奥运会（就说奥运会吧）对跨性别运动员的解释不大…牢固。或者讲奥运会的设计远远落后社会观念的版本。
